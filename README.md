# Bamf!

A pure Swift parser for ISOBMFF (ISO Base Media File Format) based formats such as MPEG-4 (.mp4), QuickTime movies (.mov), JPEG2000 (.jp2), and more.

> [!TIP]
> Why is this called Bamf!?
>
> It's my headcanon for how to pronouce BMFF
