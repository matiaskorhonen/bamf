# ``Bamf``

A pure Swift parser for ISOBMFF (ISO Base Media File Format) files such as MP4, MOV, and JPEG 2000.

Use Bamf to load media container data, walk the atom tree, and inspect atom-specific metadata.

> Tip: Why is this called Bamf!?
>
> It's my [headcanon](https://en.wiktionary.org/wiki/headcanon#Noun) for how to pronouce BMFF
